    GeoJSON

    The map demostrates the use of GeoJSON data that contains a set of polygons classified by feature types. These feature types are dynamically assigned a unique color that is helpful to quickly visually identify what resource is where on the map. The same data was used to dynamically build a legend for the map.

    The dataset containing coal resources was acquired from the British Geological Survey (link) as a shapefile that was cleaned and converted to GeoJSON using QGIS.
